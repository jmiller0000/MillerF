UNDERSTANDING CLOUD COMPUTING

The idea of computing in the cloud began in 1961 when computer scientist John McCarthy proposed:
If computers of the kind I have advocated became the computers of the future, then computing may somebday be organized as a public utility just as the telephone system is a public utility... The computer utility could become the basis of a new and important industry.

Since the mid-1990s, we have used search engines, e-mail services, social media, etc. which are all Internet-based computer utilities that built the foundation for modern-day cloud computing.

In the late 1990s, Salesforce emerged as one of the pioneers in delivering enterprise applications via a simple website, marking the rise of Software as a Service (SaaS). Amazon Web Services (AWS) was launched in 2006, offering scalable computing power as a service.

In 2006, the term "cloud computing" came to be and Amazon launched its Elastic Compute Cloud (EC2) and Simple Storage Service (S3).

From 2010 on, cloud computing gained significant traction and the introduction of Infrastructure as a Service (IaaS) and Platform as a Service (PaaS) offerings further expand cloud opportunities for businesses.

Today, cloud computing is an integral part of the modern technology landscape. Major companies such as AWS, Microsoft Azure, Google Cloud Platform, and others offer a wide array of services, including storage, computation, databases, machine learning, and more. Hybrid and multi-cloud approaches have gained popularity, allowing organizations to mix and match services from different cloud providers or combine them with on-premises infrastructure.

HOW IT WORKS

Cloud computing service providers host or maintain massive data centers that provide the security, storage capacity, and computing power to support cloud infrastructure. Clients pay for the rights to use their clouds along with an ecosystem to communicate between devices and programs (How Cloud Computing Works).

VIRTUALIZATION

Virtualization is a fundamental concept in cloud computing that allows you to create virtual representations of servers, storage, networks, and other physical machines. Virtual software abstracts physical hardware resources, allowing multiple virtual machines to run simultaneously on a single physical machine. This enables better resource utilization and efficient management of hardware, resulting in greater returns from investments. Physical servers consume electricity, take up storage space, and need maintenance. Virtualization removes these limitations and allows you to maintain and use hardware infrastructure like a web application.


ELASTICITY

Elasticity involves the ability of cloud computing systems to dynamically allocate or deallocate resources based on demand. It allows businesses to scale their resources up or down in response to changing markets.
